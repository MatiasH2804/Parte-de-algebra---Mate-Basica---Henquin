Algebra lineal - Determinantes

Calculadora paso a paso de determinantes

Carga una matriz cuadrada y obtiene el determinante con desarrollo didactico, tarjetas numeradas y ecuaciones MathJax.

1. Matriz de entrada

Admite enteros, decimales y fracciones como 3/4 o -5/2.

2. Metodo usado

Para orden 2 se aplica la formula directa:

$$\det(A)=ad-bc$$

Para orden 3 y 4 se usa expansion de Laplace sobre la primera fila:

$$\det(A)=a_{11}M_{11}-a_{12}M_{12}+a_{13}M_{13}-\cdots$$

Para orden 5 a 8 se usa eliminacion gaussiana explicada paso a paso:

$$\det(A)=(-1)^p\prod_{i=1}^{n}u_{ii}$$

3. Procedimiento paso a paso

Esperando una matriz para calcular.

Todavia no se calculo ningun determinante.
